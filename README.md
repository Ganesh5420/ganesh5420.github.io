# Ganesh Gupta Supply Chain Portfolio Website

This is a static one-page portfolio website built for a supply chain/procurement professional look and feel.

## Files

- `index.html` — Main portfolio page
- `styles.css` — Visual design, responsive layout, and animations
- `script.js` — Navigation, scroll reveal, header state, and copy-email interaction
- `Ganesh_Gupta_Resume.pdf` — Downloadable resume file used by the site
- `assets/favicon.svg` — Site favicon

## How to use

Open `index.html` in a browser. To publish it online, upload all files in this folder to Netlify, Vercel, GitHub Pages, or any static hosting service.

## Customization

Update text directly in `index.html`. Colors and layout variables are at the top of `styles.css`.
